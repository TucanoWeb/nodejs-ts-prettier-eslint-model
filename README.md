# Model base using TS, Prettier, and  Eslint

Model using the Nodejs, TS, Prettier, and Eslint.



# Get Starter



## Install Dependencies

In the root of the project, run the command:

```bash
npm install
```



## Run Project

Yet in the root of the project, run the command:

```bash
npm run dev
```
